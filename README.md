# Простая страница перенаправления (Simple Redirect Page)

## Описание

Этот репозиторий содержит один HTML-файл (`index.html`), который используется для создания промежуточной страницы с изображением перед автоматическим перенаправлением на конечную ссылку.

### Как это работает:
1.  При переходе по ссылке GitHub Pages сначала отображается изображение `fsociety` (ссылка на изображение: `https://habrastorage.org/getpro/habr/upload_files/d17/973/fa5/d17973fa5d63bf9e17294f9e0720436d.jpeg`).
2.  Через заданное время (по умолчанию 3 секунды) JavaScript на странице автоматически перенаправляет пользователя на конечную ссылку.

### Конечная ссылка для перенаправления:

## Настройка

Если вы хотите изменить изображение, время задержки или конечную ссылку, вы можете отредактировать файл `index.html` в этом репозитории.
